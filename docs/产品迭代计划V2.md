## 一、产品定位

**一句话**：小微企业（1-20人）的AI财税助理

**核心价值**：
- 不用懂会计，开口就能记账
- 不用记截止日期，AI提前提醒
- 不用手动算税，AI生成申报草稿

**目标用户**：
- 创业公司老板（不懂财税）
- 代账公司（服务大量小微企业）
- 个体工商户（没有专职会计）

### Tab 2：🤖 AI助手

**定位**：全局自然语言交互入口，所有业务都可以通过对话触发

**对话意图识别：**

| 意图 | 触发关键词 | AI行为 |
|------|----------|--------|
| 记账-收款 | "收了xx"/"收到xx"/"收入xx" | 生成收款凭证卡片 |
| 记账-付款 | "付了xx"/"支付xx"/"支出xx" | 生成付款凭证卡片 |
| 税务-查税 | "要交多少税"/"这个月税"/"Q1所得税" | 返回税种金额列表 |
| 税务-申报 | "帮我报税"/"生成申报草稿" | 打开报税Tab |
| 工资-发放 | "发工资"/"给xx发工资" | 生成工资条预览 |
| 财务-余额 | "还有多少钱"/"余额多少" | 返回当前余额 |
| 财务-报表 | "赚了多少钱"/"净利润"/"利润率" | 返回财务概览 |
| 发票-识别 | [拍照] → "帮我记账" | OCR识别 → 生成凭证 |
| 合规-预警 | "有什么风险"/"合规检查" | 返回合规建议 |

**对话显示规则：**
- 用户消息：右对齐气泡（深蓝色）
- AI回复：左对齐气泡（白色卡片）
- 凭证卡片：带确认/取消按钮
- 税务回复：带申报草稿按钮

#### 4.4 📝 合同管理（🔄 新增）

**定位**：合同全生命周期管理，从签合同到收付款到收益分析

**合同录入：**
```
合同基本信息：
  - 合同名称（必填）
  - 甲方/乙方（必填）
  - 合同金额（必填）
  - 签约日期 / 到期日期
  - 合同类型：销售合同 / 采购合同 / 服务合同
  - 付款方式：一次性 / 分期 / 按月 / 按里程碑
  - 预估毛利率（默认20%，用于收益测算）

分期付款节点：
  - 首付 X% → 日期 + 金额
  - 验收 X% → 日期 + 金额
  - 尾款 X% → 日期 + 金额
```

**合同列表UI：**
```

### Tab 5：👤 我的

**定位**：员工管理、工资发放、个人设置

**子Tab（3个）：**

#### 5.1 👥 员工（🔄 重构）

**首次进入强制引导设置：**
```
Step 1：添加员工
  - 姓名（必填）
  - 部门（选填：研发/产品/销售/运营/行政）
  - 职位（选填）
  - 身份证号（必填，脱敏存储）

Step 2：设置工资标准
  - 基本工资（必填，数字）
  - 社保个人部分（自动算，基数×8%）
  - 公积金个人部分（自动算，基数×12%）
  - 其他补贴（选填）

Step 3：确认保存
  → 生成员工档案
```

**员工列表（设置完成后）：**
```
┌─────────────────────────────────┐
│ 👥 员工（5人）      [+ 添加]    │

│ 🟦 王五  销售部    ¥7,800    [待发]│
│ ...                             │
└─────────────────────────────────┘

**员工管理操作：**
- 点员工 → 编辑工资标准
- 点"离职" → 状态改为 inactive，历史工资保留
- 点"调薪" → 修改基本工资（记录调薪日期）
